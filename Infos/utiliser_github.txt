1) ajouter un 'remote'

git remote add [remote name] [remote url]
git remote add depot git@github.com:/syffer/ProjetAlgo



2) ajouter des fichiers à prendre en compte lors des commits

git add [file1, file2, file3, ...]
git add Code Gant



3) faire un commit 

git commit -m depot 'description de ce qui a été fait'



4) pousser le commit (prendre en compte le commit)

git push [remote] [branche]
git puch depot master
